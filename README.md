# Pendulum Simulator

This project is a pendulum simulator that allows users to visualize and interact with a pendulum's motion. Users can adjust various parameters such as length, mass, starting angle, gravity, damping, time interval, and simulation speed.

## Features

- Visualize pendulum motion in real-time
- Adjust parameters to see their effects on the pendulum's behavior
- Play and reset the simulation
- Zoom controls for better visualization
